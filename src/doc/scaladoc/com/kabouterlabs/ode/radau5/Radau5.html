Radau5 interface

implicit Runge-Kutta method of order 5 (Radau IIA) for problems of the form My'=f(x,y) with possibly singular matrix M; with dense output (collocation solution).

dim

: Dimension of the ODE

funcM

: ODE solver call back

jacM

: Jacobian

massM

: Mass function; Determines whether this is a DAE

daoVar

: Index for the DAE system

params

: Parameters

config

: Configuration parameters

See also

for more information on the underlying algorithm : http://www.unige.ch/~hairer/prog/stiff/radau5.f or http://www.unige.ch/~hairer/software.html
