case class Lsodes(dim: Int, funcM: OdeFuncM[Double], jacM: JacobianFuncM[Double], params: FuncParams[Double], config: Config) extends Product with Serializable

Solves the initial value problem for stiff or nonstiff systems of ordinary differential equations (ODE). DLSODES is a variant of the DLSODE package, and is intended for problems in which the Jacobian matrix df/dy has an arbitrary sparse structure (when the problem is stiff).

dim

: Dimension of the ODE

funcM

: ODE solver call back

jacM

: Jacobian

params

: Parameters

config

: Configuration parameters

Note

: for more information on the underlying algorithm follow this link and look for odepack https://computation.llnl.gov/casc/odepack/. More info here http://www.netlib.org/odepack/opkd-sum and here (pdf) https://computation.llnl.gov/casc/nsde/pubs/u113855.pdf
