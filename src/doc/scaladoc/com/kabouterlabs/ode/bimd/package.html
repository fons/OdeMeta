    Bimd interface

    Bimd interface

    The code BIMD numerically solves (stiff) ODE or linearly implicit DAE problems of index up to 3 with constant mass matrix.

    dim

    : Dimension of the ODE

    funcM

    : ODE solver call back

    jacM

    : Jacobian

    massM

    : Mass function; Determines whether this is a DAE

    daoVar

    : Index for the DAE system

    params

    : Parameters

    config

    : Configuration parameters

    Note

    : for more information on the underlying algorithm : http://web.math.unifi.it/users/brugnano/BiM/index.html
