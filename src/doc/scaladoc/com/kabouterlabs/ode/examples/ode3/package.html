    Chemical kinetics problem taken from the dlsoda fortran code.

    Chemical kinetics problem taken from the dlsoda fortran code. This uses a full jacobian which is different from the lsodes example in Ode2

    Three rate equations :

    dy1/dt = - rk1*y1 dy2/dt = rk1*y1 + rk11*rk14*y4 + rk19*rk14*y5 - rk3*y2*y3 - rk15*y2*y12 - rk2*y2 dy3/dt = rk2*y2 - rk5*y3 - rk3*y2*y3 - rk7*y10*y3 + rk11*rk14*y4 + rk12*rk14*y6 dy4/dt = rk3*y2*y3 - rk11*rk14*y4 - rk4*y4 dy5/dt = rk15*y2*y12 - rk19*rk14*y5 - rk16*y5 dy6/dt = rk7*y10*y3 - rk12*rk14*y6 - rk8*y6 dy7/dt = rk17*y10*y12 - rk20*rk14*y7 - rk18*y7 dy8/dt = rk9*y10 - rk13*rk14*y8 - rk10*y8 dy9/dt = rk4*y4 + rk16*y5 + rk8*y6 + rk18*y7 dy10/dt = rk5*y3 + rk12*rk14*y6 + rk20*rk14*y7 + rk13*rk14*y8 - rk7*y10*y3 - rk17*y10*y12 - rk6*y10 - rk9*y10 dy11/dt = rk10*y8 dy12/dt = rk6*y10 + rk19*rk14*y5 + rk20*rk14*y7 - rk15*y2*y12 - rk17*y10*y12

    rk1 = rk5 = 0.1 rk4 = rk8 = rk16 = rk18 = 2.5 rk10 = 5 rk2 = rk6 = 10 rk14 = 30 rk3 = rk7 = rk9 = rk11 = rk12 = rk13 = rk19 = rk20 = 50 rk15 = rk17 = 100

    This problem is sparse and stiff.
