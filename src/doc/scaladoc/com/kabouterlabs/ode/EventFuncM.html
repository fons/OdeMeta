case class EventFuncM[U](eventOpt: Option[(Int, Double, Array[U], Int, Array[Int], FuncParams[U]) ⇒ Unit]) extends Product with Serializable

This is a monadic wrapper around the event function. The event function will be calleod when the constraint is met (i.e.a zero encountered). The dependent variables can be changed before the next call to the solver is made
